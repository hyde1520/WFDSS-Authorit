Creating Delegation of Authority Reports

An incident owner or editor can create Delegation of Authority Reports using one of the report templates available in WFDSS. You can copy an existing template, or create a new template. Templates must be ready and available for you to select from in the Template Selection list. You can quickly create a Delegation of Authority Report if you use an existing template. If you would like to create a new template, or copy and edit an existing template for you to use, you will have to do that first (and make it visible) so the template will be available for you to select when you create your Delegation of Authority Report.
