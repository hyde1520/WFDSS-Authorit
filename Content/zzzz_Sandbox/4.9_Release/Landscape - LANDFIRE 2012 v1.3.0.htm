Landscape - LANDFIRE 2012 v1.3.0

The LANDFIRE 2012 1.3.0 data source is the latest version of data products from LANDFIRE that incorporate disturbances, management activities, and succession through 2012. It is currently the default data source for incidents and analysis in WFDSS. Data for the update was derived from:

  • The Events Database, which is an ongoing effort consisting of 1999 or later user contributed polygon data
  • Fire mapping program information, such as Monitoring Trends in Burn Severity
  • Landsat image pairs processed using a modified version of the Multi-Index Integrated Change Analysis method to detect areas of decreasing greenness.

Data is available for the entire U.S. Visit the LANDFIRE website for additional information about LANDFIRE availability.
