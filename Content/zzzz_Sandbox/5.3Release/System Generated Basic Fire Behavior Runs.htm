System Generated Basic Fire Behavior Runs

System Generated Basic Fire Behavior runs are generated automatically by WFDSS. The intent of the runs are to produce spatial Hauling Charts for incidents of interest, which are currently defined as active incidents (not contained, controlled, or out), with a discovery date within the last 7 days, AND for which there is accepted ownership.

The Hauling Charts include the date, time, wind speed, and direction (1500 forecast winds) for the selected RAWS, and the resolution for the landscape used to generate the Basic outputs. Processing the runs is completed overnight; runs are updated each day for 7 days during the early morning hours.

Potential uses:

  • Provide a quick snapshot of modeled fire behavior for optional use in making risk-based decisions on new starts and/or prioritizing fires in multiple start scenarios.
  • Highlight the utility of the Hauling Charts as a risk assessment tool.
  • Highlight the utility of the Basic Fire Behavior model.
  • Provide fire behavior specialists with a first look at auto-generated outputs to use as a starting point for further analysis.
  • Provide managers with a tool that can be used to assess potential fire effects benefits.
  • To test functionality of auto-generated fire behavior runs for expanded/future use.
