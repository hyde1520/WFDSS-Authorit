To Restore Decision Content

  1. From the Incident list, select the incident > click View Information. The Edit Incident Information page appears.
  2. Review the information under each tab (Information, Objectives, Course of Action, Cost) to ensure that the information is complete and accurate.
  3. Select the Decisions tab. The Decisions List appears. You should already have a Pending Decision.

  4. Select the Pending Decision. The Status should read Being Edited.

    1. If the Status is not Being Edited, click Edit. The Decision Editor page appears.
    2. Click the Decision tab to exit the Decision Editor page.
    3. The Status should now read Being Edited.
  5. Select the Pending Decision, then click View Decision. The Pending Decision page appears.
  6. View and click Decision Content with a + next to Content. This allows you to review the expanded content.

    Figure 1: Decision Content

  7. Select and click a version from the expanded content to view relevant information.
  8. Review the various versions to select the appropriate version with the correct content and without errors.
  9. Click Restore.
  10. Return to the Decision tab and Edit the Pending Decision.
  11. Continue adding versioned content as needed.
