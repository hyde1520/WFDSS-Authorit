5.2 Release Introduces the Default Decision Editor

Release 5.2 will be deployed on Training 3/18/2015 and to Production 3/20/2015.

New Functionality and Enhancements

Feature/Enhancement

Description

Default Decision Editor
  • WFDSS Authors, Editors, Reviewers, Approvers and other users can now develop decision documents using the Default Decision Editor. The Default Decision Editor is an intuitive, quick, and easy way to put together a decision for an incident. The Default Decision Editor automatically opens when a Pending Decision is edited. Vertical Tabs represent each section of the decision; providing quick movement between sections. Hyperlinks in decision sections link to incomplete components to speed completion of required sections.
  • The old Decision Editor still exists in its current format and will be referred to as the Advanced Decision Editor. Users that wish to use the Advanced Decision Editor can do so by selecting the Convert to Advanced button on the Decision tab when all sections of a Pending Decision are Checked In.
  • See About the WFDSS Decision Editors for further information.

LANDFIRE 2012

  • LANDFIRE 2012 data is now available for use in WFDSS.
