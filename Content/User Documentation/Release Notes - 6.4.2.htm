Release Notes - 6.4.2

8/28/2019 - Bug Fixes for Release 6.4.2

Incident Groups

  • You can download Decisions for all decisions in the Group from the Incident Groups page, or for one or more incidents, from the Incidents Group > View Incidents page.

Analysis Naming

  • Special characters in an analysis name (backslash \ and apostrophe ') caused the decision editor to not function correctly. Restrictions on the Analysis Name text box were placed on what special characters are allowed. Per 8/28/19, the analysis name cannot contain an apostrophe or backslash; the cursor blinks ifyou try to add these characters and a message displays that these characters are disabled.

System Preferences

  • The format of WFDSS generated emails can be HTML or text.
