Data Input

The points available in WFDSS are a merged dataset containing the following files from the Federal Communications Commission (FCC) GIS page:

  • AM
  • Antenna Structure Registration
  • BRS/EBS
  • Cellular
  • FM
  • Land Mobile - Commercial
  • Land Mobile – Private
  • Land Mobile - Broadcast
  • Microwave
  • Paging
  • TV-NTSC
  • TV-Digital

Data Processing:

Attributes were stripped from this data layer. Only fields indicating the tower type, the necessary FID, and the shape field were included. Data points that were outside of the WFDSS Geographic Area Boundary were not included in dataset. The final dataset was re-projected using the "GCS" coordinate system, and the datum "North American Datum 83".
