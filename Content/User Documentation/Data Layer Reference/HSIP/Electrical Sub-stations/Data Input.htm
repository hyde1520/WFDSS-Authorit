Data Input

This feature class is from the HSIP Gold 2010 Infrastructure ArcGIS 9.3 energy geodatabase.

The following processing steps were followed to edit the original shapefile for use within WFDSS:

  1. Selected all "In Service" records.
  2. Selected records where thevalue for the attribute field"COUNTRY" equaledUnited States and US Territories.
  3. Re-projected the resulting layerto GCS NAD 83.
