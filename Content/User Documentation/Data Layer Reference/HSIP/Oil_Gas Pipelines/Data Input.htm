Data Input

This feature class is from the HSIP Gold 2010 Infrastructure ArcGIS 9.3 energy geodatabase.

The following processing steps were followed to edit the original shapefile for use within WFDSS:

  1. Selected all: " ‘ ‘ ", "ACT", "ACTIVE", "Active", "IN SERVICE" and "PABN (proposed abandonment but could be currently active)" records.

    Other attribute values that were not used:

    • A/C = ABANDONED AND COMBINED ABANDONED
    • ACTIVE - THIS PIPELINE IS IN SERVICE AND BEING UTILIZED
    • CNCL = CANCELLED
    • COMB = COMBINED
    • N/A = NOT APPROVED AT THIS TIME
    • O/C = OUT AND COMBINED
    • OUT = OUT OF SERVICE
    • PABN = PROPOSE ABANDONMENT
    • PREM = PROPOSE REMOVAL
    • PROP = PROPOSED
    • R/A = RELINQUISHED AND ABANDONED
    • R/C = RELINQUISHED AND COMBINED
    • R/R = RELINQUISHED AND REMOVED
    • RELQ = RELINQUISHED
    • REM = REMOVED
    • Advan Develop = Proposed Advanced Development
    • Early Develop = Proposed Early Development
  2. Kept only the points in US and US Territories.
  3. Re-projected the resulting layer to GCS NAD 83.
