Data Input

This feature class is from the HSIP Gold 2010 Infrastructure ArcGIS 9.3 energy geodatabase.

The following processing steps were followed to edit the original shapefile for use within WFDSS:

  1. Selected records where the value for the attribute field "OP_CAP" (Operating capacity)were greater than zero.
  2. Used electric generating units layer (HSIP 2010) andselected records where the attribute field "status" equaled "operating" (this layer has multiple points for every power plant, so use only to identify the associated plants that are in operation).
  3. Used "Select by location" to find the points from the electric power plants that are identical to the selected points in the generating units layer.
  4. Kept only the points in US and US Territories.
  5. Re-projected the resulting layer to GCS NAD 83.
