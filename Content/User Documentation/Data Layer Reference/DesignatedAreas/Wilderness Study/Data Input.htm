Data Input

The data displayed in WFDSS represent a merged layer from two original data sources (listed in the Data Information section below). Minor edits were performed on each of the source data sets to ensure a consistent format when combining them into the WFDSS version. The attributes of each layer were edited so that the WSA and Agency names were the only two attribute fields in the final merged version. These edits did not impact the original integrity of the data, and no polygons or attribute associations were changed, removed, or added.
