Overview

This layer contains features of aerial fire retardant avoidance areas delivered as part of the 2011 Nationwide Aerial Application of Fire Retardant on National Forest System Land Environmental Impact Statement. This Feature Class shows areas, provided by each National Forest who used aerial fire retardant from 2000-2010, where the aerial application of fire retardant should be avoided in order to prevent the potential of impacts to Federally listed threatened or endangered species as identified through consultation, or Forest Service sensitive species.

Data includes location of terrestrial and hydrographic areas where the application of aerial fire retardant is to be avoided. Current aerial fire retardant standards prohibit application within 300 feet of hydrographic features. Therefore, this data may contain duplicate hydrographic areas already covered by existing standards.
