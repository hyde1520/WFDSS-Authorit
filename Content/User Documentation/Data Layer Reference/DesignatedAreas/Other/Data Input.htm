Data Input

The data displayed in WFDSS represent a merged layer from three original data sources (listed in the Data Information section below). Minor edits were performed on each of the source data sets to ensure a consistent format when combining them into the WFDSS version. The attributes of each layer were edited so that the Designation and Agency names were the only two attribute fields in the final merged version. These edits did not impact the original integrity of the data, and no polygons or attribute associations were changed, removed, or added.

The list below outlines the designations (by agency) that are contained in this layer.

  • National Game Refuge and Wildlife Preserve - USFS, USFS private inholding, State
  • National Historic Area - USFS
  • National monument - BLM, USFWS, USFS, USFS private inholding, Other Federal
  • National Primitive Area - USFS
  • National Scenic Area - USFS, USFS private inholding, State Other Federal
  • National Scenic Research Area - USFS
  • National Volcanic monument - USFS, USFS private inholding, State
  • Other Congressionally Designated - USFS, USFS private inholding, State
  • Primitive National Wild and Scenic River - USFS, USFS private inholding
  • Research Natural Area - USFS, USFS private inholding, USFWS, Other Federal, State
