Overview

The Landscape Source data layer is generated "on-the-fly" in WFDSS and is based on one of two criteria:

  • If this layer is turned on while viewing the Intelligence Map, it displays the coverage of the landscape data source that is set in the user's preferences from the My Home tab.
  • When viewing this layer from the Situation Map within an incident, it displays the coverage of the landscape data source that is set on the Incident Information page.

This layer is used as a reference-only layer in WFDSS, and is meant to serve as a double-check with the user to ensure that the appropriate landscape data source is selected.
