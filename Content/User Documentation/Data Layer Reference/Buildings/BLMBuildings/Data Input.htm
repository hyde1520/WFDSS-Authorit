Data Input

All data were obtained from the BLM’s Facilities and Asset Management System (FAMS) database and converted to a shapefile based on the latitude and longitude coordinates.

Facilities with the following type attribute were retained (all other facilities were classified as campgrounds)

  • Administrative Office Space
  • Administrative Site Building Building (Propane)
  • Communication System
  • Communication Tower
  • Electrical Power Generation
  • Exterior Walls
  • Fee Collection Station
  • Fuel Distribution Fuel
  • Distribution (Propane)
  • Housing Space
  • Monument
  • Roof Building
  • Storage Tank
  • Storage Tank (Propane)
  • Warehouse Space
