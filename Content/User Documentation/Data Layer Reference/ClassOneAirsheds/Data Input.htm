Data Input

Individual shapefiles for each of three federal agencies were downloaded from the National Park Service's Air Resource Division website. These files were then merged together to create a comprehensive interagency Class I airshed boundary layer. The following agencies have Class I boundary files and were used in the creation of the national interagency layer:

  • NPS (41 class I areas) – view map
  • FWS (21 class I areas) – view map
  • USDA FS (88 wilderness class I areas) – view map
