Data Input

The data were obtained using the sources listed in the Data Information section below.

The USDA Forest Service data were downloaded as Excel spreadsheets from INFRA by Forest Service Region. The resulting spreadsheets were then mapped by latitude and longitude coordinates for each region. The nine regions were then merged into one layer representing all of the Forest Service campground locations in the INFRA database. Two fields were kept in the attribute table; Site_Name, which described the name of the campground; and Agency, which was populated with “USFS” for all records in the layer.

All data were obtained from the BLM’s Facilities and Asset Management System (FAMS) database and converted to a shapefile based on the latitude and longitude coordinates.

Facilities with the following type attribute were retained (all other facilities were classified as BLM Buildings)

  • Cabana Ramada
  • Camping Unit Developed
  • Camping Unit Semi-Improved
  • Picnic Unit
  • Recreation Area
  • Recreation Site

The edited Forest Service data were merged with the BLM data to create a national interagency dataset.
