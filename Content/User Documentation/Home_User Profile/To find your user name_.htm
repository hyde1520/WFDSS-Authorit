To find your user name:

  1. Go to the appropriate URL for the system that you want to locate an account.
  2. Click Forgot your user name? The Lookup User Names page opens.
  3. Enter an email address and click Lookup User Names. The Existing User Names with Given Email page opens and all WFDSS user accounts associated with the provided email address are listed.
  4. Select your user name from the list.
  5. Click Return. The WFDSS Login page appears, with your username populated in the User Name field.

You can now login as usual. If you continue to have issues, please contact the HelpDesk (https://iiahelpdesk.nwcg.gov/).
