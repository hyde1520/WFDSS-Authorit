My Home

The My Home tab in the application contains your user and system preferences. The My Home tab in WFDSS contains the following user-defined settings:

  • Contact Information
  • Password
  • Privileges
  • System Preferences
  • Address Book

The tasks in this section apply to everyone who uses WFDSS.

The WFDSS website (also known as the WFDSS Home page and not to be confused with My Home page within the application) is located at http://wfdss.usgs.gov, and contains news, additional resources, and other information about WFDSS. From the WFDSS application, you can access the WFDSS website by clicking the WFDSS logo in the upper left corner of the application page. You do not need a login to access the WFDSS website.
