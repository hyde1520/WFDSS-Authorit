To view group details:

  1. From My Home >select Address Book. The Address Book page appears.
  2. Select My Address Book from the Address Book drop-down list.
  3. Verify that the User Name field is blank, then click Apply Filter. The User List table changes to display the contents of your Address Book.

    Groups are displayed under the User Name column and have anicon to the right of or directly below the group name. For example, .

  4. Click the appropriate group name. The Group Update page appears.
  5. If necessary, select the appropriate filter and click Apply Filter.
  6. Review all group members and their contact, location, agency, and unit information.
  7. Click Return to access the Address Book page.
