Creating a Duplicate Account

Caution: Do not create multiple accounts unless you are a member of the development team and need them for testing. If your current access is not adequate, notify your GA Editor and request a new user role.

The Existing User Names with Given Email page displays all the user names tied to the same email address.

  • To login with an existing account, click the user name that you want to use.
  • To continue the process of creating a new account, click Create New Account.
  • To return to the Request User Account page, click Return.
