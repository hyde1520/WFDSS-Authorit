1. Choose the My Home tab. The Home page appears
  2. From the left menu, select Address Book. The Address Book page appears.
  3. Under Address Book, select WFDSS Address Book.
  4. Under Roles, select Geographic Area Editor.
  5. If necessary,use the >> button to move a specific Geographic Area from the complete list to the pick list.
  6. If necessary, use the >> button to move specific Agencies from the complete list to the pick list.
  7. Select Apply Filter. A list of the Geographic Area Editors appears that meets your filtering criteria; the list includes contact information.
