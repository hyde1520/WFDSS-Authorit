6.3 Release New Functionality and Enhancements

6.3 Release was originally deployed on Production on 11/14/2018

New Functionality and Enhancements for 5/8/2019 Release

  • WFDSS Lite
    • WFDSS Lite has been decommissioned. Remove bookmarked URLs that point to this decommissioned site.
    • Removal of links to WFDSS Lite in emails that are generated to users.
  • There are added character count downs to many of the text boxes. As you type you will see how many characters you have remaining. When you reach the limit if you try to type more a red box will drop down and tell you the limit has been reached.
  • A slightly different editor in some of the text boxes.
  • The IIA Helpdesk email address is now removed from some locations (background is that email doesn’t work for DOI and it will be changing soon anyway, the IIA Helpdesk prefers we provide users with their website as from that location you can email, chat, or call.)

New Functionality and Enhancements for 4/24/2019 Release

  • WFDSS Exchanges New Fields with IRWIN
    • The focus of the 4/24/2019 release is to upgrade to IRWIN 5.1. Most of these changes will not result in user visible changes to functionality, with the exception of the items listed below.
    • WFDSS now shares read/write access with these five fields in IRWIN: Relative Risk, Organizational Assessment, Estimate Final Fire Cost, Strategy Slider value, and the Current Decision Published Date. This allows these values to be displayed in other systems, however, none of the systems can modify these values except for WFDSS.
  • WFDSS Behavior Changes for "Certified" Incidents
    • If the records for an incident are marked as "Certified" by a final fire reporting application, then this status will be consistentbetween IRWIN participating applications. Edits will no longer be accepted to the incident information and the save buttons are grayed out. Additionally, the following message appears across the top of the incident information screen, "This incident has been Certified and is controlled by the 'Unit ID'. Please contact the FMO to update the information."
  • WFDSS Lite
    • WFDSS Lite has been decommissioned, but there may still be some references to WFDSS Lite. These will be removed in the months to come.
  • WFDSS On Mobile Devices
    • WFDSS is fully functional on mobile devices, but users on iOS mobiledeviceswill have to disable the fly-out menus. This must occurprior to using a mobile device from your workstation or laptop. Also, the fly out menus must be disabledindividually in Production and Training. For further information, see Accessibility Mode.

New Functionality and Enhancements added in 3/13/2019 Release

  • Incident Filter
    • The Incident filter displays 30,000 of the most recent incidents.

New Functionality and Enhancements added in 11/14/2018 Release

  • Subscribing to Planning Area Notifications
    • Fromthe account system preferences page, some WFDSS user roles (Viewer, Dispatcher, Author, GA Editor, National Editor, and Data Manager) can subscribe to planning area notifications for one or moreunits. Planning area subscriptions are optional, but intended to help facilitate communication between neighboring units andpromote cooperation early in the life of an incident and decision-making process.
  • MAP Displays
    • Pending M.A.P.s display as dotted lines. Activated M.A.P.s, those that are included in published decisions, display with asolid line. Deactivated M.A.P.s display with a dashed line. Once a decision is published, pending M.A.P.s become active and display with other, previously activated M.A.P.s and appear as solid lines.
    • When you exclude an M.A.P. and publish a new decision, excluded M.A.P.s are 'deactivated' and display with dashed lines in map displays.
  • Password Change Update
    • All users, including Administrators and National Editors,must change their passwords every 60 days. WFDSS sends 10-day and 3-day warning email alerts to help you avoid account lockout. If you fail to change your password prior to expiration, your account is locked and you must call the Help Desk.
  • Updated Helpdesk Information
    • If you cannot find an answer to your question in the WFDSS Help, you can call the Help Desk at 1-866-224-7677 (Toll Free), +1 616-323-1667, or navigate to https://iiahelpdesk.nwcg.gov/. Fire Applications Help Desk provides level 1 technical support for WFDSS.
  • Predictive Services Significant Fire Potential
    • Topics containing references to this service have been removed.
  • Requesting an Account
    • You may now include *usda.gov email addresses when requesting a WFDSS account.
  • Fire Behavior Requests
    • Fire behavior requests are monitored only by GA Editors.
