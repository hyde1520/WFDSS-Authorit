To view user role requests:

  1. Choose the Administration tab. The Administration page appears.
  2. Click Show Role Request. The table displays the role requests available to you.

    Note: If your list is longer than a page, use the User List Filter to search by agency or a specific user.

  3. Select the user you want to view roles for.
  4. Click Edit User. The Edit User Role page appears.
  5. Review the user's information and justification.
  6. If appropriate, select the User Role(s) requested by the user.
  7. Click Save. The user's role will be updated when they log into WFDSS.

Note: If a user is currently logged into WFDSS when you change their role, they need to log out, close and restart the web browser, then log back in for the role change to take effect.
