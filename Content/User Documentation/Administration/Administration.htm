Administration

Note: Before starting tasks available to the left in the Administration tab, ensure you understand how each task works and its consequences.

Users with Geographic Editor or Administrator roles have access to the Administration tab. The following functions are available to both of these users:

  • View Role Requests
  • Edit User Roles when there is a role request
  • Group emails to all users that currently appear in the list (Excluding anonymous and disabled users)

Administrators can also:

  • Edit user profiles and/or user roles if a user calls for assistance.
  • Reset Passwords when users cannot reset passwords themselves from the login page or have been locked out.
  • Broadcast System Messages from within the application to all users.
