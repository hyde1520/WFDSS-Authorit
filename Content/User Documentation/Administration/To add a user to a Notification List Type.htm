To add a user to a Notification List Type

  1. From the Administration tab, select Notification from the left menu. The Notification Types list appears.
  2. Click the radio button to the left of the appropriate Notification Type.
  3. Click Open. Displays the appropriate Notification List.
  4. Click Add. A new row displays at the top of the list.
  5. Click the radio button to the left of the empty User ID, User Name, and Email Address row.
  6. In order to send notifications to the email address and phone number stored in the contact information, enter the User name and user ID. In order to receive notifications to an alternate email address, enter the User name, and alternate Email address. Do not enter both a user Id and an Email address into the same Notification line. In order to receive notifications to both the email identified in the Contact information, and an alternate email address, two notification entries must be created.
  7. Click Save to add the user to the Notification List, or click Cancel to not save the user information.
  8. Click Return to go back to the Notification Type page.
