To reset a user's passwords:

  1. Verify that the user you are unlocking is a valid WFDSS user.
  2. Choose the Administration tab. The Administration page appears.
  3. To more easily find the account you want to unlock, filter the list by role, geographic area, agency, or name.
  4. Click Apply. The User List table displays the users who meet your filter criteria. If the list is empty, try modifying your criteria.
  5. Select the user account you want to reset passwords for.
  6. Click Edit User. The Password Reset page appears.
  7. Enter a new password.
  8. Re-enter the new password to confirm it.
  9. Click Reset Password.

The system generates an email letting the user know that their password was reset. The message DOES NOT contain the new password. The Help Desk or Administrator calls the user with a new password or clicks Send Access Key on the Administration > User List > Password Reset page. The user is now able to login to WFDSS, and needs to change their password the first time they login.
