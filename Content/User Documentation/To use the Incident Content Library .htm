To use the Incident Content Library

  1. From the Incident List, select the incident you want.
  2. Click View Information. The Incident Information appears.
  3. Click Menu to display the fly out menu. This step is not necessary if you have Enhanced Accessibility enabled and menu options are anchored on the left.
  4. Select Incident Content Library. Under the Content sub-tab, the incident content information displays.
  5. Click the incident content you want to view and it displays in the incident content pane on the right.
  6. To return to the Incident perspective, click the Menu sub-tab and select the appropriate option to continue working.
