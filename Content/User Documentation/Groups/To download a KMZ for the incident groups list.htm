To download a KMZ for the incident groups list

  1. From the Incident Groups tab, select the incident group you want to download a KMZ for and then click View Incidents. The list of incidents within that group displays.
  2. Click the Generate KMZ link located above the list of incidents. The KMZ Download for Incident Groups List page appears.
  3. Select the items to include in the KMZ download, and then click Continue.
  4. Entera Download Label of your choice.
  5. Click Submit Request. The screen returns to the Incident Groups List page and a message displays at the top of the screen that the KMZ request has been sent for processing.
  6. Navigate to the My Home tab > My Activities to access the downloaded KMZ file.
