Viewing an Incident Groups Map

Users can display a map view for all incidents in an Incident Group. From the map view (Click 'View Group Map' from the Incident Groups List Filter page), you can view planning areas, fire perimeters, and points of origin for the group. On the map, click 'Incidents' in the layer switcher to see each Incident. Continue clicking the plus sign to review analysis information for each incident in the group.

The mapping tools displayed at the top of the map page will help you navigate. Above the mapping tools is a Bookmark button that allows users to save their current map displays, so they will be available the next time they log in to WFDSS. There is also the option to share any Bookmark as a link with other users whose privileges allow them to view the map context.

The three tabs on the left of the page (Menu, Map, Info) contain information relevant to the group situation and group analysis perspective.

Menu tab. Depending on your user role, you can:

  • Return to the Incident Groups List.
  • Click on Group Analyses to see Analysis List for all analyses that are part of the Incident Group.
  • Access the Landscape Editor to create landscape editor rules.
  • View specific information about or create a landscape file.
  • Upload shapes.
  • View the Group Analysis Report for the current group analysis.
  • View incident Contact Information for the group.
  • Manage Incident Privileges.
  • Download a RAWS KMZ for the group.
  • View Fire-Related Links.

Map tab. Depending on your user role, you can:

  • View information for a variety of incident groups map layers.
  • Drawor copy analysis or incident shapes.
  • Draw and save a Landscape Extent.

Info tab. Depending on your user role, you can:

  • Draw and save an Analysis Extent.
  • Determine the Latitude and Longitude for any point on the map.
  • View, Copy, or Download Feature Information.
  • Review data related to the situation, such as the Smoke Dispersion Forecast,Strategic Objectives,Fire Weather Forecast,and Hourly Weather Forecast.
  • Capture group map images.
