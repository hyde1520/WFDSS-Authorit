Creating Group Filters

You can create and save Incident Groups filters as well as create temporary Incident Groups filters. Each filter you create is added to your Incident Groups Filter List. Use Incident Groups filters to find groups that you own or other public groups. When you no longer need a filter, you can clear them.

You can create a filter for, or filter on, one or more of the following parameters:

  • Group Name
  • Owner
  • State
  • Group Type
