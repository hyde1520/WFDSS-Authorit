Flame Act Report

The Flame Act report is no longer used officially in fire business, though some users still rely on the Flame Act Report function to indicate when Type 1 or Type 2 teams are desired for particular fires. The Flame Act Report automatically generates when:

  • An Organization Assessment indicates a Type 1 or Type 2 Incident Management Team is recommended to manage an incident,
  • The Incident Size or Latest Perimeter size is 300 acres or more,
  • The Incident Type is Wildland Fire,
  • The Responsible Unit Name is Federal, and
  • The Jurisdictional Agency at Point of Origin is not blank.

The report is archived in the WFDSS Reports tab and generates information including:

  • Incident Size
  • Most Recent Perimeter Size
  • Organization Assessment and Recommended Organization

Once generated, users who've subscribed to the Flame Act notification will receive an email that the report is available for review from the WFDSS Report tab.
