WFDSS User Documentation Overview

The documentation for WFDSS is primarily in the form of online help, training webinars, white papers, and the WFDSS website. Here's where to go to find information:

Table 1: Types of WFDSS Documentation

Documentation Type

Purpose

Information Found Here

Online Help

Provides users with instructions on working with the WFDSS application, as well as to provide reference information about fire behavior modeling.
  • Task-based step-by-step instructions on using the application
  • Decision Reporting and Business processes
  • Reference information about fire behavior
  • Description of roles and responsibilities
  • Field descriptions
  • Glossary terms

WFDSS 101 - Training Series

The lessons in this series are intended to familiarize users with WFDSS and be completed as a suite, and in numerical order, or as stand-alone lessons. Each lesson contains one or more short exercises that will teach you how to use WFDSS tools, as well as ideas for incorporating the tools into your workflow.

  • Roles and responsibilities
  • Workflow
  • Decision Reporting and Business processes

White Papers

Provides background information about topics related to WFDSS and the Decision Reporting process, as well as to explain policies related to fire management.
  • Policy explanations
  • Helpful hints for GIS-related activities
  • Background information that you need to understand when using WFDSS for decision reporting
  • Other information related to WFDSS

WFDSS website Page

Provides the public with information about the project, and to provide resources and news to WFDSS users.
  • Training schedules and registration information
  • Links to the application (Production and Training)
  • Related Resources
  • Contact information for the Help Desk

WFDSS SupportVideos YouTube Channel

Provides training videos
