Navigate to an Analysis

  1. Navigate to the analysis by doing one of the following:

    • Choose Incidents tab > select the incident > click View Analyses > select the analysis.
    • Choose Analyses tab > select the analysis.
    • From within the incident, choose Incident Analyses from the left menu > select the analysis.
    • From the Situation Map, choose Menu tab > Incident Analyses > select the analysis.

    Note: Use the temporary filters or create permanent filters to help you more easily find the analysis you are looking for.
