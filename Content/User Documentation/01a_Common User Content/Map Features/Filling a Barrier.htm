Filling a Barrier

When working with Analyst Assisted STFB, NTFB and FSPro fire behavior models, Fire Behavior Specialists can fill a barrier from the Model Information page. A Filled Barrier is a barrier shape with unburnable fuels in its interior. Fill Barrier is the default. If an analyst chooses to use a barrier with burnable fuels in its interior, the analyst can select Do Not Fill.
