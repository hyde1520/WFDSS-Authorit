To download a single fire perimeter:

  1. From the Incident List, select the incident you want to download perimeters for.
  2. Click Assess Situation. The Situation Assessment page appears.
  3. From the left pane, expand Incident layer > Fire Perimeters. The available fire perimeters appear in the list.
  4. Select the perimeter you want to download, then click the expansion arrow to the right of the shape name. The Layer Details appear below the selected shape.
  5. Click the Download Shapes icon. The File Download window appears.
  6. Click Save. The Save As window appears.
  7. Navigate to where you want to save the perimter ZIP file.
  8. Click Save. The ZIP file is downloaded to the selected folder.
