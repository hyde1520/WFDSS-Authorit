To fill a barrier

  1. Choose one of the following methods for navigating to the Model Information Menu Option:

    • From the Analysis List page, select the analysis that you want to select analysis shapefiles for > click View Information.
    • From within the model, select the Model Information Menu Option
  2. Locate the Analysis Shape Files section.
  3. Click the radio button before Fill Barrier to make the interior of the barrier unburnable. Click the radio button before Do Not fill to make the interior of the fire burnable.
  4. Click Save.
