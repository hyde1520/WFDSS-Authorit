Unit Maps

The Unit Map, available from the Data Managementtab, allows users to quickly view a selected administrative unit’s outline, or associated Unit Shapes that may include FMUs, Strategic Objectives, Management Requirements and/or Other Unit Shapes. The Unit Map is especially helpful for Data Managers, as it allows for immediate viewing of Unit Shapes recently uploaded (eliminating the need for users to navigate away from the Data Management tab to a different map display) and drawing of unit shapes. It also assists with spatial fire management planning.

The intelligence, incident, and analysis map layers are not available for viewing from the Unit Map, because the purpose of the Unit Map is to assist with data management. For this reason, you cannot create, copy, or manage incident and analysis shapes from the unit map. All other map layers are available for viewing.
