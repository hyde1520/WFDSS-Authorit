Shape Privileges by User Role

Each WFDSS User Role is associated with privileges that determine who can upload, draw, manipulate, and copy shapes. The following shapes cannot be modified when a decision is in the Review and Approval process: Management Action Points, Objective Shapes, Planning Area, and Fire Perimeters.
