To download all fire perimeters for an incident:

  1. From the Incident List, select the incident you want to download perimeters for.
  2. Click View Information. The Edit Incident page appears.
  3. From the left menu, choose Download Perimeters. The Download Incident Perimeters page appears.
  4. Type a label for your download. For example, enter Four Mile 7-15-2010.
  5. Click Download. File Download window appears.
  6. Click Save. The Save As window appears.
  7. Navigate to where you want to save the perimter ZIP file.
  8. Click Save. The ZIP file is downloaded to the selected folder.
