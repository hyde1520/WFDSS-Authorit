Downloading Fire Perimeters

You can download fire perimeters three different ways:

  • A single fire perimeter
  • All perimeters for an incident
  • All final fire perimeters displayed in the current incident list

NOTE: When a zip contains more than one shapefile, only the largest shapefile (in terms of acres) is loaded.

Before you can download fire perimeters, the following conditions must be met:

  • A unique Fire ID must be assigned to each incident you are downloading perimeters for.
  • Perimeter shapes must exist for each incident you are downloading perimeters for.
