Filters

WFDSS filters enable you to customize the views for the following pages:

  • Incidents
  • Objectives
  • Course of Action
  • Analyses
  • Decisions
  • Reports
  • Administration
  • Intelligence

If there are filters that your National and/or GA Editors are using for the incident, analysis, and intelligence pages, you can request they share that filter with you.

Note: If you set up a filter for analyses and then want to set up a similar filter for incidents or for use on the intelligence map, you must re-create the filter on the appropriate page.
