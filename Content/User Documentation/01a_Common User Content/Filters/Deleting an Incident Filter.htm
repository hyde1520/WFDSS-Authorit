Deleting an Incident Filter

You can delete a permanent incident filter, whether it is a default or one that you've created, by selecting Manage Filters. You delete a temporary filter when:

  • You create a new incident filter.
  • Select an existing filter from the Filters drop-down list.
  • Log out of WFDSS.
  • You use the Incident Name quick search tool, locatedabove the incident list.
