Editing an Incident Filter

After you create an Incident Filter, you might find that you need to modify it. For example, if you are an author, you may want to filter for all the incidents on your unit that have a decision. As a result, your permanent filter will require modification each time you have a new incident that requires a decision. Deleting an incident filter is an option, also.
