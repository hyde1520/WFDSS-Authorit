Analysis Filters

WFDSS allows you to set up temporary and permanent analysis filters based on the following parameters

Note: Any analysis filters you set up are available for use with analyses only. If you want to set up a similar filter for incidents or for use on the intelligence map, you must re-create the filter on the appropriate page.

Other Filters

  • Analysis Type
  • Geographic Area
  • Request Date
  • Completion Date
  • Analysis Status
  • Unit ID

Analysts

  • User Name
  • Geographic Area
  • Agencies

Incident Authors

  • User Name
  • Geographic Area
  • Agencies

Incident Names

  • Incident Name
