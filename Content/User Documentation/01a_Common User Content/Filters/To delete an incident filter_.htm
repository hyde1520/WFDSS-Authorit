To delete an incident filter:

  1. From the Incidents List page, click Edit Filter List. The Incident Filter List page appears.
  2. From Incident List Filters, select the filter you want to delete.
  3. Click Delete. A message appears asking if you are sure you want to delete the filter.
  4. Click OK. The filter is removed from the list.
  5. Click Return. The Incident List page appears.
