About the WFDSS Website Home Page

You can navigate to the WFDSS Website Home page from any page in the application by clicking on the Wildland Fire Decision Support System icon in the upper left corner of the screen.

The WFDSS Website Home page provides general, publically available information about the WFDSS application. The website should not be confused with the WFDSS My Home page, which allows a user to manage his/her own account and user preferences.

From the WFDSS website, you can do the following:

  • Access a variety of training material in PDF format or short training videos (Home > WFDSS Training).
  • Access WFDSS-related supplemental information for NWCG training courses (Home >NWCG Training WFDSS Courses).
  • Access the WFDSS online help (Home > WFDSS Help).
  • View white papers, glossariesand various publications about WFDSS related topics (Home > Related References).
  • View the latest information about WFDSS releases andrelease notes (Home > What's New in WFDSS).
  • Access login pages for both WFDSS Production and Training (Home > Sign In to Production or Training).
  • Quickly access the most popular offerings on the WFDSS Website (Home > Hot Picks).
  • Initiate a Google search of the WFDSS website and/or the world wide web.
  • Accessinformation about Data used in WFDSS(Home > Data).
  • Request a WFDSS account.
  • Access the HelpDesk and the National Fire Decision Support Center via website or telephone.
