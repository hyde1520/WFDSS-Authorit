Source Types

Source Types refers to how the shape files were derived. There are 8 source types.

  • Drawn / Ground: Shape hand-drawn by physical observation on the ground
  • Drawn / Air: Shape hand-drawn by flying over a feature in an aircraft
  • Drawn / Satellite: Shape hand-drawn by using satellite imagery, such as MODIS
  • GPS/ Ground: Shape GPS-tracked by physical observation on ground
  • GPS / Air: Shape GPS-tracked by flying over a feature in an aircraft
  • Infrared: Shape derived from infrared imagery
  • Other: Shape derived from another method not listed
  • Unknown: Shape derived from unknown source
