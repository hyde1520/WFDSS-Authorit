Analysis Name

A sequence of descriptive characters chosen to identify a particular analysis.  For example, "14d 500f 0826p OakFlat" would identify a 14-day FSPro run that simulates 500 fires, uses a perimeter dated August 26 and uses the OakFlat RAWS for weather and wind data.

Underscores and other punctuation are not recommended and an analysis name can be changed any time before an analysis is submitted.

Descriptive names become important when multiple analyses exist for an incident and are limited to 48 characters.
