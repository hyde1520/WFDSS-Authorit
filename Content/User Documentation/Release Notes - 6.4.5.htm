Release Notes - 6.4.5

5/14/2020 - Bug Fixes for Release 6.4.5

Account Management

  • Modified the column name from "90 Days of Inactivity" to "60 Days of Inactivity" in the MS Excel Spreadsheet downloaded from the User Security List.

Analysis

  • Fixed a bug resulting in a 404 error when viewing FSPro Values at Risk.
  • Fixed a bug causing a missing (GAP) record in the Hourly Weather when a Basic, Near Term, or Short Term Analysis was copied.

Application Support

  • Updated a System Check utility to notify support staff if the IRWIN Service is disconnected.
  • Updated the IRWIN Service so that it does not use the same architecture as any of the other services that process messages to improve support capability.
  • Updated the Weather Station Utility to fix a bug causing the weather-related emails to consistently report failures for stations that are not used in WFDSS.
  • Fixed a bug causing the Utility to loop at times, which prevented the ERC data from updating.
  • Fixed a bug causing an IRWIN incident to insert into WFDSS without an IRWIN ID.
  • Corrected logging of PDF generation to improve future debugging.
  • Improved logging to help debug issues with Decision Failures including Certificate Error.

Calendar

  • Adjusted the calendars used throughout the application to begin January 1, 2006, rather than only the previous ten years.

Clean Up/Maintenance

  • Removed WFDSS Lite links in all WFDSS environments.
  • Removed the IRWIN.V6-enabled properties and code from the application and IRWIN Service Broker.

Cost

  • Updated the values used for calculating the Stratified Cost Index (SCI) from the 2014 to the most current (2017) data.
  • Updated the Cost Estimator Spreadsheet (2018) that is available for download from the Incident Cost Page.

Data Layers

  • Corrected the Sage Grouse Layer to correctly match the legend.
  • Fixed a bug preventing Aquatic Retardant Avoidance from appearing in Values Inventory and Values at Risk.
  • Updated the Feature Information for the Jurisdictional Agency Layer to include the Landowner Kind and Category.
  • Fixed a bug preventing uploaded historic perimeters to be added to the corresponding Historic Layer if the incident’s Jurisdictional Agency was County & Local.
  • Fixed a bug resulting in an error message when uploading a historic perimeter.
  • Updated the Horse and Burro Layer so the colors align with the legend.
  • Shortened the labels displayed on maps for BLM Lands to improve readability.
  • Corrected issues with the following data layers: Responsible Agency, Building Clusters, Wilderness, Communication Towers, Jurisdictional Agency, and Wild & Scenic Rivers.

Data Management

  • Fixed a bug causing Data Managers to receive a 500 error when navigating to the Strategic Objective List if their Unit associated with their user profile was not a valid unit.
  • Fixed a bug resulting in an error message when loading multiple management requirements shapes or using the replace shape feature.
  • Removed triplicate text in an Alaska Statewide Management Requirement.

Decisions

  • Added a vertical scroll bar to the accordion pane of the default decision editor so that users can access all content when Broadcast and System Messages present are present.
  • Fixed a bug causing the Image Containing all Management Actions Points to not be included in a copied Decision.
  • Fixed a bug that prevented users from including carriage returns while editing content in some text boxes.
  • Fixed a bug causing the Incident Objectives and Requirements text to not wrap.
  • Fixed a bug causing the published Benefit and Strategy Slider images from the View Decision to reflect the most recent images and not the published images.
  • Fixed a bug resulting in a 500 error when scrolling through the View Decision for decisions published before June 2017.
  • Fixed a bug causing the Benefit Slider Comment to overflow the text box when Viewing a Decision.

Feedback

  • Corrected an issue where a Feedback’s response text was deleted if an attachment was added.
  • Fixed a 504 Gateway timeout error that occurred when managing Feedbacks. Fixed a bug preventing Feedback attachments from opening when submitted in Internet Explorer.

Incidents

  • Fixed a bug causing the Management Action Points list to be inaccurate after a user downloaded the Management Action Point Shapes.
  • Fixed a bug causing the Assessment Tab to display the risk tables for the wrong incident if the incident that was being viewed had a planning area but no risk tables.
  • Fixed a bug causing the Federal Admin. Areas to be inventoried in the Values Inventory and Values at Risk.
  • Fixed a bug causing a null pointer error when utilizing the Objectives Tab for incidents in Alaska.
  • Added tracking of the Pushing to IRWIN or Pulling from IRWIN to the Incident History List.
  • Changed the Values Inventory/Values at Risk to display the Landowner Categories instead of the Jurisdictional Agencies.
  • Fixed a bug preventing the download of the Incident Objective List when an incident is locked (has an out date, is out for review, or the user does not have edit privileges).

IRWIN

  • Updated code to allow the Landowner Kind and Landowner Category to be set in IRWIN when the Jurisdictional Agency, for a WFDSS created incident, at the point of origin could not be determined.
  • Fixed a bug causing the Latitude and Longitude to not update in IRWIN when a user modified the values in WFDSS.
  • Fixed a bug preventing the Calculated Acres in IRWIN from being updated when a perimeter was uploaded in WFDSS.

Support Functions (Administrators)

  • Updated Broadcast Message text for warnings so they do not interfere with the automated tests.

Support Functions (National Editors)

  • Fixed a bug causing a 500 error when trying to view an issue from the Irwin Issue List and communication with IRWIN was disabled.
  • Added a vertical scroll bar to the WFDSS/IRWIN Comparison table in Internet Explorer to prevent the data from being cut off.
  • Decommissioned the indexing of Decisions that allowed for Decision searches by National Editors. The decision search feature will work for existing decisions but newly published decisions will no longer be indexed.
  • Fixed a bug resulting in a 500 error when searching for specific terms in the decision search feature.
  • Fixed a bug causing the IRWIN Issue Description in the IRWIN Issue List for an Issue to sometimes show the error description for a prior issue.
  • Fixed a bug causing the IRWIN Issue List to incorrectly report an issue when communication to IRWIN was disabled and a non-Wildfire Incident in WFDSS that was not linked to an IRWIN Incident was updated.
