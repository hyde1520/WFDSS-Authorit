To copy feature information:

  1. From an Incident or Analysis map, select the Identify tool.
  2. Click the point of interest.
  3. From the left pane, choose the Info tab, then click Feature Information. The Feature Information popup window appears.
  4. Select a layer from the Query Layer drop-down list.
  5. Click Query Layer. A message notifies you if the layer does not intersect your point of interest, and positive query results are displayed. Layers Queried keeps track of layers previously queried for that point.
  6. Click the Copy icon next to the feature you want to copy. The Feature Copy popup window appears.
  7. Type a Shape Label and select the Shape Type from the drop-down list.

  8. If you select Fire Perimeter as the Shape Type, enter the appropriate Fire Perimeter Input Parameters:

    • Effective Date
    • Effective Time
    • Source Types
    • Comments
    • Mark the Update Incident size checkbox if you want to update the incident size
  9. Click Copy. The feature is copied.
