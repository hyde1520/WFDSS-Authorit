Intelligence Gathering

The WFDSS Intelligence tab allows users to gather information relevant to their regions, including the Intelligence map, reports, and other data. The mapping tools displayed at the top of the page allow you to work with the map and create an incident if you are an Author or Dispatcher. If you are an Author, you can create an incident group using the polygon tool. Above the mapping tools is a Bookmark button that allows Dispatchers, Authors, and Editors to save their current map settings on the Intelligence map so they will be available the next time they login to WFDSS. There is also the option to share any bookmark as a link with other users whose privileges allow them to view the map context.

By adding the spatial layer Active Planning Area in the System Preferences tab, you can display nearby active planning areas on the Intelligence map.

The three tabs on the left of the page contain information relevant to an incident:

Menu tab. Depending on your user role, you can:

    • Review/modify the Intelligence map.
    • Generate Summary Reports.
    • Set Intelligence Preferences.

Map tab. Depending on your user role, you can:

    • Manage incident filters for the Intelligence map.
    • View map layers in the LayerSwitcher.

Info tab. Depending on your user role, you can:

    • Select location/layer/feature information to query.
    • Review data related to the incident such as the Smoke Dispersion Forecast, Strategic Objectiveswithin XX radius of point of origin,Fire Weather Forecast, and Hourly Weather Forecasts.
    • Create Incident Groups using the Polygon tool.
