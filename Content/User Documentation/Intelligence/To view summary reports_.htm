To view summary reports:

  1. Access the Summary Reports page:

    • From the Incident list, click View Summary Reports.
    • From the Intelligence map, select the Map tab > Summary Reports.

    The Summary Reports page appears.

  2. Select the Report Type.
  3. Select the Calendar Year.
  4. If you selected geographic area for the report type, select which geographic area you want to view the summary report for.
  5. When you are done viewing the report, click Return. The page refreshes to either the Intelligence map or the Incident List, depending on how you accessed the report.
