About the ERC Stream (FSPro)

The ERC Stream populates when an analyst generates (and saves) ERC Classes on the ERC Stream Menu Option. The stream contains the:

  • Average ERC for each date listed.
  • Standard deviation (a measure of dispersion or how widely spread the values are from the mean) for the average daily ERC values.
  • Current ERC value.

The value in the current ERC column displays a "-1" when there is no ERC value from the RAWS/NFDRS. From that point on, assuming the -1 is on, or one day away from the simulation start day, FSPro obtains the ERC value from the forecast. After the specified forecast period (number of days selected) or if no forecast data is used, FSPro uses weather and wind data from the WFDSS-generated time series.
