About Maps

WFDSS has six map views:

  • Intelligence Map : Provides a large-scale, national overview of incidents (based on your filter settings and user preferences). The default view is your assigned geographic area, but you can change the view in your user settings. You can use the to create an incident or enter the latitude/longitude on the Info tab. Dispatchers and fire managers commonly use this view to assess wildfires from a regional or national perspective.
  • Situation Map : Allows users to assess the situation for a selected incident, throughout the duration of the incident. The Situation map is available from the Incidents tab and is the most commonly used map view in WFDSS. It is the primary map used when creating, editing, and updating a decision document.
  • Analysis Map : Similar to the Situation map, the Analysis map allows fire analysts to assess the landscape when creating a fire behavior analysis. It is available from the Analyses tab and is the primary map used by analysts when creating, editing, and updating an analysis.
  • Landscape Map: Allows fire analysts to view the landscape file that's been generated for an analysis and verify that the LCP Editor Rules applied to the landscape are working correctly. Landscape files used for analysis can also be viewed on the Analysis Map. Using the 'i' tool in all other map displays queries the default landscape data source without edits.
  • Results Map: Allows fire analysts to view the results of their fire behavior analyses in Review status. It also allows all users the opportunity to view accepted and completed fire behavior analyses results for any incident. The Results map for an analysis is accessible from the Analysis List.
  • Unit Map: Accessible from the Data Management tab, the Unit Map allows users to quickly view a selected Unit’s outline, associated FMUs and unit shapes. The Unit Map will be especially helpful for Data Managers, as it allows immediate viewing of unit shapes recently uploaded, and will eliminate the need for users to navigate away from the Data Management tab to a different map display. It also assists with spatial fire management planning.
