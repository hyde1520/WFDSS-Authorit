About General FSPro Input Parameters

General FSPro Input Parameters is a required section of the FSPro Information page that includes four primary fields. The section contains:

  • Analysis Name,
  • Number of Fires,
  • Number of Days, and
  • Analysis Start.

Fire Behavior Specialists evaluate these Parameters and choose appropriate values/settings or, accept the defaults without modification. You can modify content in these fields up until an FSPro run is initiated. After a run begins, the system locks the content in the fields and they can no longer be edited.
