About Run FSPro Status

This section of the FSPro Information page lists the input requirements still needed by FSPro before you can initiate a run. This section differs from the Status column in the Analysis List, that refers to the status of the run overall (for example, Assigned, Ready for Review, Complete etc.).

Run FSPro Status is only visible and applicable when a run is Assigned to a Fire Behavior Specialist. Once all input requirements are met, the status reads "An FSPro Analysis can be run." The Run FSPro Status section disappears as soon as an analysis run begins.
