About the My Home page

The My Home page is comprised of four sub-tabs that each displays different customized information about your recent WFDSS incidents, activities, map bookmarks, and other system messages.

  • My Incidents
  • My Activities
  • Map Bookmarks
  • Messages

The left-hand menu on the My Home page also allows you to manage and make changes to the following:

  • Contact information
  • Password
  • Security Question
  • Roles
  • System Preferences
  • Address Book

The tasks in this section apply to everyone who uses WFDSS.

The My Home page should not be confused with the WFDSS Website. The My Home page allows you to manage user-specific information about your account within the WFDSS application, while the WFDSS Website provides general information about the overall WFDSS application and is open to the public.
