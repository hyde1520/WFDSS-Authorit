Editing and/or Copying an Existing Report Template

Unless users have special privileges mentioned below or the user created a template themselves, they will need to copy a template from the template list before they will be able to view or edit that template.

Incident Owners, Data Managers, GA Editors, and National Editors who create a Report Template can edit that template when the template is marked as visible. Data Managers and GA Editors can edit templates marked visible that are created by anyone in their geographic area. National Editors can edit templates marked visible that are created by anyone in the nation. When the Report template is marked personal, only the template creator can edit the template. If you have the privileges, you can also edit the properties, edit the content, copy, and/or delete a Report Template marked as visible.

Incident Owners, Data Managers, GA Editors, and National Editors who have privileges can a copy a template from the Template List. This involves copying the template from the Template List and renaming it, adding tooltips, and a description if needed. Data Managers, GA Editors, and National Editors can edit the Agency and Unit. When the Report template is marked personal, only the template creator can copy, rename, and modify the template.
