Guidelines for Creating Template Types

WFDSS enforces a variety of rules about what templates types are available and what roles are authorized to create report templates.

This table summarizes the template types.

Table 1: Description of Template Types

Type

Description

National

The template is appropriate for use within any incident.

National and Agency

The template is appropriate for use by a given agency within any incident.

Geographic Area

The template is appropriate for incidents within a given Geographic Area.

Geographic Area and Agency

The template is appropriate for use by a given agency for incidents within a given Geographic Area.

Unit

The template is appropriate for incidents within a given Unit.

Personal

The template is for use by the individual who created the template.
