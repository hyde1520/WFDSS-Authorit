To edit text in a Report Template

  1. From the Reports List click Create, or click Edit Report Templates from the Template List. The Template Editor opens.
  2. Select the section of the Report Template you want to edit the existing text.
  3. Position the cursor in the text box and begin typing and formatting.
  4. After you have completed typing for each section of the template, click Save.

When you are done adding content, you can navigate back to the Report List or Template List to Check-In the Report Template.
