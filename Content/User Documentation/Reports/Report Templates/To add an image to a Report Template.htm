To add an image to a Report Template

  1. From the Reports List click Create, or click Edit Report Templates from the Template List. The Template Editor opens.
  2. Select the section of the Report Template you want to add the image.
  3. Navigate to the Images Folder in the lower left corner of the Template Editor.
  4. Select the image to insert.
  5. Place your cursor in the Template Editor pane (large area on the right side of the Template Editor page) where you want the image to appear.
  6. Click the WFDSS Clipboard icon to insert the content. (The icon is on the upper left side of the Template Editor).
  7. Click Save. The image is now part of the Report Template.

When you are done adding the image, you can navigate back to the Report List or Template List to Check-In the Report Template.
