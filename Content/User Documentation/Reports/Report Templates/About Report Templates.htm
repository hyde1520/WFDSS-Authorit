About Report Templates

After an Author, Data Manager, GA Editor, or National Editor selects an Incident from the Incident List, they can create a template from Reports tab > Edit Report Templates > Template List, by clicking the Create button at the bottom of the Template List. If other templates already exist, they can be accessed from the Template List. If templates are marked National, Agency, Geographic Area, or local unit, they are available for review and editing if you have appropriate privileges. Without the proper privileges, templates can be copied to create a new editable template. Templates can be personal, national, or specific to a geographic area, agency or local unit. Personal templates can only be seen by the person who created the template, while national geographic area, agency or local unit templates are available for all eligible users to copy, edit and use.

Each report template is pre-populated with the incident's current Objectives and Course of Action.

Template users can:

  • Create a new Report Template
  • Check in or check out a report template
  • Edit and/or copy an existing Report Template
  • View Information/History about a Report Template
  • View a Report Template report
  • Convert a Report Template into a .pdf file
  • Publish a Template Report
  • Delete a Report Template

You should review the guidelines for creating template types and what roles are authorized to create each template type before working with report templates.
