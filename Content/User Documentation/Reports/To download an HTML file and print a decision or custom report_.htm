To download an HTML file and print a decision or custom report:

  1. From the Incident list, select the incident > click View Information. The Edit Information page appears.

  2. Select the decision or report that you would like to download:

    • Choose Decisions tab > select the published decision you want to download.
    • Choose the Reports tab > select the available custom report that you want to download.
  3. Click Download to save the decision/report to your computer or network drive. The File Download dialog box appears.
  4. Click Save. The Save As dialog box appears.
  5. Navigate to the folder where you want to save the decision/report, then click Save. The files get saved to the folder you selected.
  6. Extract the contents of the ZIP file. The contents of the ZIP folder are extracted into a new folder.
  7. Double-click on the HTML file to open it.
  8. Choose File > Print. The Print Options window opens.
  9. Select your printing opens, then click Print. The document prints to the designated printer.
