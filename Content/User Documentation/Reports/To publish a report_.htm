To publish a report:

  1. From the Incident list, select the incident > click View Information. The Edit Information page appears.
  2. Choose the Reports tab. The Reports list appears.
  3. From the Report List, select the report that you want to edit.
  4. Click Publish. A message appears asking if you are sure that you want to lock all content.
  5. Click OK. The report's status changes to Published and the Download button becomes available.
  6. Click Download to save the report to your computer or network drive. The File Download dialog box appears.
  7. Click Save. The Save As dialog box appears.
  8. Navigate to the folder where you want to save the report, then click Save. The files get saved to the folder you selected.
