To edit map images as landscape-oriented reports

  1. From the Incident tab, select the incident for which you generated a report and click View Information. The Edit Incident Information page appears.
  2. Select the Reports sub-tab.The Report List appears.
  3. Select the report name you want to edit from the Report List.
  4. Click Edit. The Report Editor opens for the report.
  5. Use the Text Editor to modify the text.
  6. Click Save after edits are complete.
  7. Click Report List to return to the Report List page.
