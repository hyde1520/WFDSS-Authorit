To create custom reports:

  1. From the Incident list, select the incident > click View Information. The Edit Information page appears.
  2. Select the Reports tab. The Reports list appears.
  3. If you wish to create a new report,click Create.
  4. Enter a Report Name.
  5. Choose Custom Report Type to automatically format your report, and then click Create.
  6. Review the Report Name for accuracy, if edits to the Report Name are necessary, make those edits and then click Save.
  7. Click Return to navigate back to the Report List.
  8. From the Reports List you canview, edit, create and download a draft for or publish a report. You can edit reports that show a status of Available. Once reports are published, they can no longer be edited.
