To delete map images as landscape-oriented reports

  1. From the Incident tab, select the incident for which you generated a report and click View Information. The Edit Incident Information page appears.
  2. Select the Reports sub-tab.The Report List appears.
  3. Select the report name you want to delete from the Report List.
  4. Click Delete. The Delete Confirmation box appears.
  5. Click OK to permanently delete the report and return to the Report List, or click Cancel to return to the Report List.
