Creating New Reports

An incident owner or incident editor can create new reports that contain a variety of decision and/or incident content that a user selects. WFDSS offers three types of reports for users to create:

  • Custom reports contain decision and/or incident content that a user selects.
  • Management Action Point Reports contain one or more M.A.P.s and associated images, as well as any other decision and/or incident content a user wants to add.
  • Delegation of Authority reports that are developed based on a template you create, or copy from the list of available templates.

Once a report is created, it is saved to the Reports List where it can be accessed for editing and eventually publishing. Once published, report content is permanently locked and can be downloaded and printed. Report content that contains content not yet published in a decision can be be downloaded as a draft .PDF for viewing or sharing.

If you need to modify the name of a report after its been created, you can do so by clicking View Information/History on the Decisions page. The name of a report cannot be modified once it is published.
