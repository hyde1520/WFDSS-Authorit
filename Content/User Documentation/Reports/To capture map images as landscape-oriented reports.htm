To capture map images as landscape-oriented reports

  1. Go to one of the following:
  2. Select Situation > select the Incident Name and click Assess Situation. From the left menu, click the Map sub-tab and select the appropriate layers from the LayerSwitcher.
  3. Select Analysis > select the Incident/Analysis Name and click View Information.The information for the Incident/Analysis Name appears. From the left menu click Analysis Map and select the appropriate layers from the Layerswitcher.
  4. Click to capture the image. For the best image view, do not use full screen mode to capture the image unless you want the image compressed to fit the .pdf format.
  5. Enter the Image Label and any appropriate Description.
  6. Enter the Report Name not using any special characters except an apostrophe or backslash.
  7. Click Capture Image.A message saying that the application is generating a report containing that image appears at the top of the page.
  8. Select Reports from the Incident sub-menu. A few seconds after capturing the image, the report name displays in the Report List.
  9. Click the report name in the Report List and click .PDF. The Report Daft PDF page appears.
  10. Click Download Draft to review the draft report.
  11. Click Open on the file download box. The draft .pdf report opens in landscape mode.
  12. Add text to the image using the Text Editor if needed.
  13. Click Save to save the changes when you are done.
