To view M.A.P.s:

  1. From the Incident list, select the incident you want to view M.A.P.s for.
  2. Click View Information. The Edit Incident Information page appears.
  3. Choose Mgmt Action Points. The Management Action Points List page appears.
  4. Select the M.A.P. you want to view details for.
  5. Click View. The View Management Action Point page appears.
