To delete a management action point:

  1. From the WFDSS Incident List, select the incident you want to edit management action points for.
  2. Click View Information. The Edit Incident page appears.
  3. Select Mgmt Action Points. The Management Action Points List page appears.
  4. Select the Management Action Point you want to edit.
  5. Click Delete. A message appears asking if you are sure you want to delete the M.A.P.
  6. Click Yes to delete the M.A.P.
