To create incident obj. shapes:

  1. From the Incident List, select the incident that you want to create incident obj. shapes for.
  2. Click Assess Situation. The Situation Map for the incident appears.
  3. On the map, draw or upload the shape that you want to use. When you are finished, the shape should appear yellow with a Temp name.
  4. From the left pane, choose the Map tab. The May Layer Tree appears.
  5. Click the next to Incident Obj Shapes. The Layer expands to display the Name field.
  6. Type the name for the shape.
  7. Click Save. The Associate Incident Objectives/Requirements with Shape page appears.

From here, you can associate the shape to one or more incident objectives or requirements.
