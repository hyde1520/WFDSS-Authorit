To delete incident objectives and requirements:

  1. From the Incident list, select the incident you want to delete objectives/requirements for.
  2. Click View Information. The Edit Incident page appears.
  3. Select Objectives. The Objectives page appears.
  4. Select the objective/requirement you want to delete.
  5. Click Delete. A message appears asking you if you are sure you want to delete.
  6. Click Yes. The selected objective/requirement is removed from the list.
