To transfer incident ownership to a Group:

  1. From the Incident List, select the incident you want to transfer ownership.
  2. Click View Information. The Edit Incident Information page appears.
  3. Click Transfer Ownership. The Transfer Ownership page appears.
  4. Select My Address Book from the Address Book drop-down list. Your personal WFDSS address book appears.
  5. From the User List, select theradio button next to the group you want to assume ownership of the incident.
  6. Click Transfer Ownership. The Edit Incident Information page reappearsand displaysthe name of the new group owner.
