To filter incident privileges

Note: You must create an incident before you can filter incident privileges.

  1. Choose Incidents > select the incident> click View Information. The Edit Incident Information page appears.
  2. From the left menu, choose Incident Privileges. The Grant Incident Privileges page appears.
  3. If necessary, click User List Filter to expand the box.
  4. Select the Address Book and Roles you want to filter on.
  5. Select each Geographic Area and Agency that you want to restrict the filter to, then click >> to add the selection.
  6. Mark which types of Enabled Users you want to filter the user list on.
  7. Enter User Name for a person you are searching for.
  8. Click Apply Filter. The Grant Incident Privileges page refreshes and displays the filtered results in the User List table.
  9. Repeat as necessary to find the users you want to grant incident privileges to.

You can now grant incident privileges to the appropriate users in your filtered list.
