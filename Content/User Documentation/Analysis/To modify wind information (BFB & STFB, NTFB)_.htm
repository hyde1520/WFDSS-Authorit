To modify wind information (BFB & STFB, NTFB):

  1. Navigate tothe analysis that contains the wind information you want to modify by doing one of the following steps:

    • Choose Analyses tab > select the analysis > click View Information.
    • If you are already in the analysis, select the (Analysis) Information menu option from the list on the left.

    The General Information page appears for the selected analysis.

  2. Scroll down to the Wind Information section and, if necessary, click the down arrow to display the Wind Information.
  3. Verify the Speed and Direction and modify if necessary. The page refreshes after each entry.
  4. Select the Wind Type. The page refreshes.
  5. Click Save.

You can continue to modify analysis parameters until the analysis is run.
