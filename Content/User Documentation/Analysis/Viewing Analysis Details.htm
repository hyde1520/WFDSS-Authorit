Viewing Analysis Details

While an analysis is being developed, any WFDSS user can view saved model inputs. Edits on the other hand, can only be made by a Fire Behavior Specialist or a user that has been granted analysis privileges by the primary analyst. Once an analysis is accepted as complete, any WFDSS user can view its details. Analysis details can be viewed these ways:

  • As an analysis report that can be accessed by clicking:

    • View Report from the Analysis List,
    • Report menu option from the left hand menu from within an analysis, or
  • By scrolling through the model inputs for an analysis.
