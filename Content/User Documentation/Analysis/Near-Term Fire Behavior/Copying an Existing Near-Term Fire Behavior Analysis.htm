Copying an Existing Near-Term Fire Behavior Analysis

Fire Behavior Specialists can copy analysis parameters for Near-Term Fire Behavior Analysis. Copying an existing analysis allows you to do the following:

  • modify your parameters without having to re-enter everything
  • easily revert to an earlier calibrationto update ignitions and barriers

CAUTION: After copying an analysis, verify that all analysis settings and notes are correct for their new analysis. Be sure to use a different name for the new analysis.

Before copying and running the NTFB, make sure that you have a current ignition file and current barriers file.
