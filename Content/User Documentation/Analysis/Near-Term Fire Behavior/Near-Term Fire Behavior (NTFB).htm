Near-Term Fire Behavior (NTFB)

Fire Behavior Specialist roles can modify the General Information parameters for Near Term Fire Behavior (NTFB) analyses that they are running. The following parameters can be modified in an Near-Term Fire Behavior Analysis:

  • General Parameters

    • Analysis Name
    • Conditioning Days
    • Spotting Probability (<+60 min)
    • Crown Fire Method
    • Foliar Moisture Content (%)

  • Analysis Shape Files

    • Ignition File
    • Barriers File (Fill or Do NOT Fill)

  • Station Information

    • Station (Weather Station)
    • Fire Danger Rating Graph

For NTFB analyses that are accepted and marked complete for incidents and intended for the Incident Risk Console (RisC), an option appears at the bottom of the FSPro Information page allowing Fire Behavior Specialists to indicate whether an analysis is Intended for Incident Risk Console. Click Yes to include the incident in the Incident Risk Console or No to leave the analysis only in the WFDSS application. The default allows completed analyses to be included when the incident is currently in the Incident Risk Console.
