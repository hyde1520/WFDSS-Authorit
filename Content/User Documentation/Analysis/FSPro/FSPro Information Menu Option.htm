FSPro Information Menu Option

The FSPro Information page hosts a variety of fields that a Fire Behavior Specialist must populate before they can start an FSPro analysis (selecting a barrier is the only input not required). The page also contains information about the status of analysis inputs prior to running the model. This page has three sections:

  • General FSPro Input Parameters,
  • Analysis Shape Files, and
  • Run FSPro Status.

You can modify the content in these fields up until an FSPro run is initiated. After a run begins, the system locks the content in the fields and you can no longer edit them.

For FSPro analyses that are accepted and marked complete for incidents and intended for the Incident Risk Console (RisC), an option appears at the bottom of the FSPro Information page allowing Fire Behavior Specialists to indicate whether an analysis is Intended for Incident Risk Console (RisC). Click Yes to include the incident in the RisC, or No to leave the analysis only in the WFDSS application. The default allows completed analyses to be included when the incident is currently in the RisC.
