To view ERC percentiles

  1. From the Analysis List, select the FSPro analysis for which you want to view the ERC percentiles.
  2. Click View Information. The General FSPro Information page appears.
  3. From the left menu, select ERC Classes. The FSPro ERC Classes page appears.
  4. If no ERC classes have been created, generate ERC classes.
  5. Select View Percentiles below the ERC classes table. The ERC Percentiles window appears.

Note: If the percentile table is blank and you are looking at ERC percentiles in an assigned analysis, generate ERC classes again. If the percentile table is blank and you are looking at ERC percentiles in an analysis in review, complete or rejected status, copy the analysis so that you can generate ERC classes and view ERC percentiles in the copied analysis. You cannot generate ERC classes in an analysis that is in review, complete, or rejected status.
