To download an LCP file:

  1. From the Analysis List page, select the analysis that you want to create a landscape for.
  2. Click View Information. The General Parameters page appears.
  3. From the left menu, select Landscape. The Landscape File page appears.
  4. Click Download LCP File. The File Download window appears.
  5. Click Save. The Save As window appears.
  6. Browse to the folder on your hard drive or network where you want to save the file.
  7. Click Save. The ZIP file is saved to the folder you selected. This might take a few minutes, depending on your connection speed and the size of the file.
