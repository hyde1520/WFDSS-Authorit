To add rows to the Winds Matrix:

  1. From the Analysis list, select the incident/analysis name that you want to run an FSPro Analysis for.
  2. Click View Information. The General FSPro Information page appears.
  3. Select Winds from under the FSPro Information menu on the left-side of the page. The FSPro Winds page appears.
  4. Click Add Row. Add additional rows as needed by repeating this step.
  5. Click Save Winds.
