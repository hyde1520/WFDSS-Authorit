Analysis for Fire Behavior Specialists

As a Fire Behavior Specialist, you can run a Basic Fire Behavior (BFB), Short-Term Fire Behavior (STFB), Near-Term Fire Behavior(NTFB), or FSPro analysis on any incident.

Only Fire Behavior Specialists can run a NTFB or FSPro Analysis.

You can also copy existing analyses and then make appropriate changes before running a new analysis.

Note: You receive an email when your analyses are ready for review. To go directly to the analysis, click the hyperlink provided in the email.
