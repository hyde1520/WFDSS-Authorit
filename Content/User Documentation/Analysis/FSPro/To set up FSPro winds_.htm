To set up FSPro winds:

  1. From the Analysis List page, select the FSPro analysis for which you want to modify the winds.
  2. Click View Information. The General FSPro Information page appears.
  3. From the left menu, select Winds. The FSPro Winds page appears.
  4. Verify that the selected RAWS is the one most appropriate for the analysis.

  5. Set the time filter, start date, end date, start hour, and end hour.

    • The month filter is typically set to one month before and after the ignition, which is the default.
    • The hour filter should only include the time when the fire advances (the burning period.)
  6. Select the Winds Type. For Weighted Winds, acalm weight and calm threshold is automatically calculated once the wind matrix is generated.You don't need to modify the Calm Weight or Calm Threshold unless you edit the matrix.
  7. Click Generate Wind Matrix. After a few seconds, the generated matrix appears. The bins display a percentage, expressed as whole numbers with two decimal places (e.g., 1.54% appears as 1.54).

  8. Click Wind Rose. A graphical represenation of the winds appears. You can use the rose to answer the following questions:

    • Is the wind direction logical?
    • Are the wind speeds realistic based on current conditions and historical observations?
  9. Click Save Winds.
  10. Document your decisions about winds in the FSPro analysis notes.

You can now modify the wind matrix to fit the analysis parameters you want to include. (See Editing the Wind Matrix for more information.)
