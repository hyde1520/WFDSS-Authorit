FSPro Fire Behavior Results Downloads

Output from FSPro is available for download to view in other programs such as ArcMap and Google Earth. Files labeled as degrees, meters, or ASCII grids can be opened in ArcMap. Files labeled as KMZs can be opened in Google Earth. Greyed-out downloads indicate that the item wasn't used in the analysis and is therefore not available, or that the user does not have sufficient privileges to view the item. Available downloads include:

  • Output (degrees)
  • Output (meters)
  • Ignition (degrees)
  • Ignition (meters)
  • Barrier (degrees)
  • Barrier (meters)
  • ASCII Grid
  • Fire Size Histogram
  • KMZ
  • Event Coverage
  • Daily Acres
  • Average Time Grid
  • Arrival Day Distribution
  • Suppression Effect
  • Ensemble Perimeters
  • Analysis Input
