Viewing the Analysis Report

The analysis report is a valuable resource to analysts, as well as to users who want to further document a given analysis. The analysis report can give insight into why particular inputs were used, as well as what the limitations and assumptions were for a particular analysis. It also helps clarify how an analysis result was obtained. This information is especially valuable as conditions and personnel change over the course of an extended incident.

The Analysis Report is available for all fire behavior analyses and displays the following:

  • Incident Information
  • Analysis Information
  • RAWSinformation
  • Weather observations used fordead fuelfor weather and wind streams
  • Landscape information (including landscape editor rules)
  • Notes saved by the analyst

FSPro reports display the following additional information:

  • FSPro Results (Fire Sizes and Histogram)
  • FSPro ERC Classes information (RAWS used for ERCinformation, filter, classes and ERC Percentiles)
  • FSPro ERC Stream and weather forecast data
  • FSPro Winds (RAWS used for winds, filter, wind matrix. and windrose)

It is a good idea to distribute this report with any hard-copy maps displaying the analysis results.
