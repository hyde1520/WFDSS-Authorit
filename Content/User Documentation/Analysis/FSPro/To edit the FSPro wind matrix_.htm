To edit the FSPro wind matrix:

  1. From the Analysis List page, select the FSPro analysis for which you want to modify the winds.
  2. Click View Information. The General FSPro Information page appears.
  3. From the left menu, select Winds. The FSPro Winds page appears.
  4. Review the time filter.
  5. Review the calm weight and calm threshold.
  6. Click Wind Rose to review a graphical representation of the existing wind matrix.
  7. To add a row, click Add Row. WFDSS generates a row at the bottom of the matrix.
  8. To edit a bin, select the number you want to modify, then enter the new number.
  9. To delete a row, click Delete Row. The bottom row of the matrix is removed.
  10. When satisfied, click Save Winds and document your decision in the analysis notes.
