To edit a landscape file:

  1. From the Analysis List page, select the analysis that contains the landscape parameters you want to modify.
  2. Click View Information. The General Parameters page appears.
  3. From the left menu, select Landscape. The Landscape File page appears.
  4. Verify that the latitude and longitude of the landscape extent are correct and make modifications if necessary (you may have to navigateto the Analysis Map to draw a new landscape extent).
  5. Verify that the resolution is correct or enter a new value.

    Note: Finer resolutions require longer processing times.

  6. Verify that the landscape data source is correct, or select a different source.
  7. Select either 40 or 13fuel models.

    Note: Only the LANDFIRE landscapes have both 40 and 13 fuel model sets.

  8. Verify that the landscape extent exists and the landscape editor rules are correct.
  9. Click Save.
  10. Click Create LCP File. WFDSS generates the landscape file.

You can now view the landscape. You must re-create the Landscape File if you make additional changes to the landscape editor rules or the landscape parameters.
