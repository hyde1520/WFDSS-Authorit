Editing the Landscape File

A Landscape File requires edits when its attributes do not reflect actual conditions on the ground, or if its overall parameters were set up incorrectly. Fire Behavior Specialists can edit the landscape in the following ways:

  • Create or import landscape editor rules to modify Landscape File attributes, such as fuel model, canopy cover, canopy base height, stand height, or canopy bulk density.
  • Modify landscape parametersto alter the landscape extent, resolution, data source, or fuel models used.

You need to re-create the Landscape File every time you modify the landscape editor rules or landscape parameters.
