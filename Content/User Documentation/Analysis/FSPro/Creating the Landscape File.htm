Creating the Landscape File

Fire Behavior Specialists can create a landscape file to use in Basic Fire Behavior (BFB) and Short-Term Fire Behavior (STFB), Near-Term Fire Behavior (NTFB), and FSPro fire behavior models. The following rules apply:

  • You must create a Landscape File before running the fire behavior models.
  • You must re-create the landscape file anytime you apply newlandscape editor rules or modify the Landscape File'sparameters (Landscape Extent, Resolution, etc.).
  • Landscape Files are created automatically in the Automated BFB and STFB models.

  • Due to processing limitations, the maximum size of the landscape file varies by type of fire behavior analysis:

    • FSPro: 2.5 million cells
    • Basic and Short-Term (Basic and STFB): 1 million cells when using gridded winds; 2.5 million otherwise
    • Near-Term (NTFB): no limitation
  • Additional processing limitation for STFB and NTFB: each cell on the chosen landscape must be conditioned prior to the model run. You are affected by this when you have a very large landscape and little fire growth. The conditioning time for a large landscape can quadruple the model run time.
