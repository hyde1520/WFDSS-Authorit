To Download FWX Weather Files (FSPro)

  1. From the Analysis List, select the analysis that you want to download FWX weather files for and click View Information.
  2. From the left menu, select the ERC Classes Menu Option. The FSPro ERC Classes page appears.
  3. Select a RAWS from the drop-down list or accept the default.
  4. Click Save.
  5. Click Download FWX and follow the directions to save the FWX file to your hard drive.
  6. Follow instructions in the FFP User's Guide to import FWX files.
