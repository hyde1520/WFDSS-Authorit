To generate a landscape critique:

  1. From the Analysis list, select the analysis you want to generate a landscape critique for.
  2. Click View Information.The General Information for the selected analysis appears.
  3. From the left menu, select Landscape.User-defined parameters for the landscape file appear, including the landscape extent, resolution, landscape data source, and fuel model.
  4. Click Generate LCP Critique. A message appears saying that the critique is being processed. Depending on the size of your LCP file, it might take a minute or two.
  5. Select Landscape again to refresh the page. A message appears at the top of the page informing you that the landscape critique was successfully generated.

You can now download the LCP critique.
