ERC Classes Menu Option (FSPro)

Clicking on the ERC Classes menu option brings you to the ERC Classes page, which hosts a variety of information that helps a Fire Behavior Specialist select a weather station, generate ERC Classes, and calibrate overall weather parameters for an FSPro run. ERC Class page information includes:

  • Station Information
  • Date Filter information
  • ERC Correlation Parameters
  • ERC Classes

The content available for viewing and editing on the ERC Classes page varies based on the status of the FSPro run being viewed and the role of the user accessing it. ERC Classes page content can be:

  • Viewed for FSPro runs of any status by users with any WFDSS role (viewer, author, etc.) but not edited.
  • Edited for FSPro runs in Assigned status by the Fire Behavior Specialist who owns the analysis and WFDSS users granted edit privileges for that analysis.
