Granting Analysis Privileges

As a Fire Behavior Specialist, once you accept an analysis, you can assign analysis privileges. Assigning privileges facilitates teamwork on complex analyses and enables users without an analyst role to view outputs. For example, you might want several people to review the analysis before it is accepted (FBAN/LTAN).

Only those personnel assigned analysis privileges can view the analysis before the results are accepted or completed, Fire Behavior Analysts, and WFDSS Administrators, who can view any analysis. Once the analysis is completed, all users can view the results.
