Downloading a Landscape Critique

You can open or save a landscape critique file, which provides summary information regarding a landscape file for the WFDSS fire behavior tools. Any WFDSS user can download a landscape critique, or LCP critique, for accepted fire behavior analyses. However, only Fire Behavior Specialists can download LCP critiques for analyses that have not been completed or accepted.

Note: You must generate a landscape critique before you can download it. A landscape critique is downloaded as a PDF file.
