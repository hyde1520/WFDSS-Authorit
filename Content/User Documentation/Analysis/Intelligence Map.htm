Intelligence Map

Any WFDSS user can use the intelligence map for the following tasks:

  • to create an incident using the map view or data entry fields.
  • to review current incidents in the selected region (defaults to your assigned GA)
  • to view values and reference layers
  • to evaluate values at risk
  • to review unit shapes available

The Intelligence Map page displays a map centered on the GA you initially selected in contact information when you created your WFDSS account, as well as three tabs in the left pane: Menu, Map, and Info. You can change the default map view in the Intelligence Preferences.

Note: You can find out about the Menu tab items by pausing over each item and reading the hover text explaining the function of each.

When you click on the Map tab and the + next to each item (to expand them), you can display many informative layers on the map. If you can't see a reference or value layer in the list, go to your system preferences and add it. These layers are periodically updated in WFDSS.
