To download a landscape critique:

  1. From the Analysis List, select the analysis you want to download a landscape critique for.
  2. Click View Information.The General Information for the selected analysis appears.
  3. From the left menu, select Landscape.User-defined parameters for the landscape file appear on the page, including landscape extent, resolution, landscape data source, and fuel model.
  4. Click Download LCP Critique. The Open or Save File dialog box appears.
  5. Click Save.
  6. Navigate to the folder you'd like to save it to, and assign a different file name, if needed.
  7. Click Save.

The downloaded landscape critique file is now available for you to open and peruse. You need Acrobat Reader® to open the file. If you don't have it installed, you can do so from the Adobe website.
