To delete an analysis note

  1. From the Analysis List, select the analysis that you want to delete a note for.
  2. Click View Information. The Analysis Information page appears.
  3. From the left menu, select Notes. TheAnalysis Notes page appears and displays a drop-down menu, a text box, and a list of existing notes.
  4. To remove an individual note, select the radio button in front of the note you want to delete and click Delete.
  5. To delete all notes, click Delete All. A message appears asking if you are sure you want to delete.
  6. Click OK.
  7. Click Return. Your previous analysis page appears with the deleted note(s) removed.
