To create analysis for an incident:

  1. Log in to WFDSS and navigate to your incident.
  2. Select View Information. The Information page appears.
  3. Select Incident Analyses from the left hand menu. The Analysis List shows for your incident.
  4. Select the type of New Analysis you want to create.
  5. Click Create Analysis for Incident.
  6. Complete the general parameters for the model and click Create.

Note : For additional information, see On-line Help for the appropriate model.
