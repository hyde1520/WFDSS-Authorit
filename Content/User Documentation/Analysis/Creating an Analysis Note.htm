Creating an Analysis Note

Fire Behavior Specialists and Incident Owners/Editors (that have been given analysis edit privileges) can add notes to analyses associated with their incidents. You can add notes to the following pages:

  • Analysis Information (for all analyst assisted fire behavior models)
  • ERC Classes, ERC Stream and Winds pages (FSPRO only)
  • Analysis Results (map)
  • Landscape (map)
  • Landscape Editor

If you copy an analysis, the notes are also copied. You can add additional notes and edit or delete existing notes at any time. Text boxes have character limits which can be easily exceeded when copying formatted text from outside the application.
