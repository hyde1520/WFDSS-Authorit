To Import Analysis Notes

  1. From the Analysis List, select the analysis that you want to import a note for.
  2. Click View Information. The Analysis Information page appears.
  3. From the left menu, select Notes. TheAnalysis Notes page appears.
  4. Click Import Notes. The Analysis List for the incident appears.
  5. Select the analysis that contains the notes you want to import.
  6. Click Import Analysis Notes. The Analysis Notes page opens and displays the imported notes.
  7. Click Import Rules to import additional rules when appropriate.
