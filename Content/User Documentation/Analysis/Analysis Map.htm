Analysis Map

The Fire Behavior Specialist (FBS) uses the analysis map for the following tasks:

  • to draw and save the landscape extent for a fire behavior analysis
  • to draw and save landscape masks used for modifying landscape data on the Landscape Editor page
  • to draw or upload barriers to the landscape
  • to draw or upload ignition shapes
  • to view and overlay analysis features on the landscape generated for the analysis

Analysis results cannot be viewed on the Analysis Map. Analysis results assigned complete status can be viewed from the Results Map or from the Situation Map.

Note: You can find out about the Menu tab items by pausing over each item and reading the hover text explaining the function of each.

When you choose the Map tab and click the + next to each item (to expand them), you can display many informative layers on the map. If you can't see a reference or value layer in the list, go to your system preferences and add it. These layers are periodically updated in WFDSS.
