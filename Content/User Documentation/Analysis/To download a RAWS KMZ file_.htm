To download a RAWS KMZ file:

  1. From the Analysis List page, select the analysis you want to download a RAWS KMZ for.
  2. Click View Information. The General Analysis Information page appears.
  3. From the left menu, choose RAWS KMZ. The Download File dialog box appears.
  4. Click Save. The Save As dialog box appears.
  5. Navigate to the folder where you want to save your file on your network or hard drive, then click Save. The file is saved to the designated folder.

Once file is downloaded to your computer, you can double-click the file to open it in GOOGLE Earth.
