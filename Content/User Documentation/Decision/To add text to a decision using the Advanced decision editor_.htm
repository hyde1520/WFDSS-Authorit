To add text to a decision using the Advanced decision editor:

  1. Choose the Decisions tab
  2. For decisions that have not been converted to the Advanced Decision Editor, ensure all sections of the decision are Checked In and select Convert to Advanced.
  3. Select the Pending Decision > click Edit. The Advanced Decision Editor page appears.
  4. From the Decision Content Tree, select the Content you want to edit. The content appears in the text editor pane.
  5. Place your cursor where you want to add text, then start typing.
  6. Click Save. (You want to save frequently, especially if you are adding a lot of text.)

When you are done editing the decision content, you can navigate back to the Decision tab to Check-In the Pending Decision.
