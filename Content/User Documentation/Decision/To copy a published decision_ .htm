To copy a published decision:

  1. From the Incident list, select the incident > click View Information. The Edit Information page appears.
  2. Select the Decisions tab. The Decision List page appears.
  3. Select the published decision you want to copy.
  4. Click Copy.The decision is copied andthe Decision List page reappears with the copy as the pending decision.
  5. Select the pending decision, then click the + to expand the list to see each component of the decision.
  6. Edit the decision as needed.
