To filter FMU or SO codes:

  1. Choose Data Management > FMU/SO Codes. The FMU/SO Codes page appears.
  2. Select the Geographic Area, Agency, and Unit that you want to find an FMUcode for.
  3. Scroll down to the Filter section and expand the box if needed.
  4. Check or uncheck the Never Activated, Activated, and/or Inactive check boxes.
  5. Click Apply Filter. The FMU/SO Codes page refreshes and displays the filtered results.

If nothing appears in the FMU Code table after you apply the filter, modify your filter parameters to make them less restrictive.

Note: SO codes do not have a filter.
