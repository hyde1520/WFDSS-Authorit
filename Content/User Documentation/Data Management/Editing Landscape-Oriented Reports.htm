Editing Landscape-Oriented Reports

Data Managers can edit the text information in a landscape-oriented report (that has been developed from a captured map image) until the report is published. If the map capture featured in the report becomes outdated, you must generate a new landscape-oriented report to generate an updated, landscape-oriented image. 

All reports are edited from the Reports tab. The only reports that can be deleted are those that have not been published.
