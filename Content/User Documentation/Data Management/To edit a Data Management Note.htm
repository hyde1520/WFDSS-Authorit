To edit a Data Management Note

  1. Select Data Management. The Strategic Objectives and Management Requirements page appears.
  2. Select the Notes sub-tab. The Data Management Notes page appears.
  3. Select the note you want to edit from under the Notes for Unit <xxx> heading, and then click Edit. The Data Management Note editing page appears.
  4. Modify the note text as needed not exceeding 4000 characters..
  5. Click Save. The Note is saved in the database.
  6. Click Return to return to the Strategic Objectives and Management Requirements page or add, edit, or delete other notes.
