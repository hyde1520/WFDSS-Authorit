Activating an FMU or SO Code

Data Managers can activate FMU codes. Spatial data is not required to use FMU codes, however, it is required to use Strategic Objective (SO) codes. The SO codes are automatically activated when the spatial data for the SOs are loaded.

Activating an FMU code makes it available for linking to the text data for strategic objectives and/or management requirements.Strategic objectives and/or management requirements associated with that FMU or SO Code can be included in decisions.

Before activating an FMU code, verify that he FMU code is correct and associated with the right GA, agency, and unit.
