To delete a Data Management Note

  1. Select Data Management. The Strategic Objectives and Management Requirements page appears.
  2. Select the Notes sub-tab. The Data Management Notes page appears.
  3. Select the note you want to delete from under the Notes for Unit <xxx> heading.
  4. Click Delete. The Delete Confirmation box appears.
  5. Click OK to permanently delete the note and return to the Notes page, or click Cancel to return to the Notes.
