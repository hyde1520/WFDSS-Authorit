To edit an FMU or SO code:

  1. Choose Data Management > FMU/SO Codes. The FMU/SO Codes page appears.
  2. Select the Geographic Area, Agency, and Unit that you want to edit FMU or SO codes for.
  3. Scroll down to the FMU or SO Code list for the selected unit.
  4. Select the FMU or SO Code that you want to edit.
  5. Click Edit. The Edit FMU/SO page appears.
  6. Change the Description.
  7. Click Save. The FMU/SO Code page reappears and displays the modified FMU or SOcodein the list.
